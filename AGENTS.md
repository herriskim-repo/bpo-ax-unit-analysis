# AX BPO Project

This repository supports BPO AX pilot automation.
## Language Rules

모든 설명은 한국어로 작성한다.
비개발자도 이해할 수 있게 설명한다.
기술 용어는 영어 병기 가능.
보고서는 한국어 기준으로 작성한다.
Security Rules
실제 회사 데이터 업로드 금지
고객사명 원본 저장 금지
GitHub에는 코드/문서만 저장
실제 데이터는 Google Workspace 내 관리
Business Rules
인건비율 = 인건비 / 매출
KPI 목표 = 62% 이하
분석 단위 = 사업부 → UNIT → 고객사
Reporting Rules
경영진 보고 기준으로 작성
복잡한 기술 용어 최소화
Action 중심 코멘트 작성
AX Direction

목표는 AI 서비스 구축이 아니라,
운영 규칙과 분석 노하우를 조직 자산으로 전환하는 것이다.
