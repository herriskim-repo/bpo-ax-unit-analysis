# AGENTS.md

## Language Rules

- 모든 설명은 한국어 기준으로 작성한다.
- 비개발자도 이해할 수 있게 설명한다.
- 기술 용어는 영어 병기 가능하다.

---

## Security Rules

- 실제 회사 데이터 업로드 금지
- csv/xlsx 원본 commit 금지
- GitHub에는 코드와 문서만 저장
- 실제 데이터는 Google Workspace 내 관리

---

## Business Rules

- 인건비율 = labor_cost / revenue
- KPI 목표 = 62% 이하
- 분석 단위 = 사업부 → UNIT → 고객사

---

## Reporting Rules

- 경영진 보고 기준으로 작성
- Action 중심 코멘트 작성
- 기술 설명 최소화
- 운영 인사이트 우선

---

## AX Direction

목표는 AI 서비스 구축이 아니라,
운영 규칙과 분석 노하우를 조직 자산으로 전환하는 것이다.

즉:

개인 경험
→ 운영 규칙
→ AGENTS.md / Skills
→ 조직 지식

구조를 만드는 것이 목적이다.

---

## Codex Behavior Rules

Codex should:

- prioritize maintainability
- keep Python scripts simple
- avoid over-engineering
- generate readable reports
- optimize for business users instead of developers
